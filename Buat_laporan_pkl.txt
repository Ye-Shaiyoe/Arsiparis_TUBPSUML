Ini adalah website persuratan BP Suml yang dibuat untuk memenuhi tugas PKL SMK Alfalah(persuratan.bp.suml.com)

Website ini menggunakan tools:
Framework utama: Laravel 12
Database: MySQL
Frontend: Bootstrap, TailwindCss, ChartJs, AlpineJs, GSAP, AnimeJS.
Backend: Laravel 12, Breeze, Node, Redis, Rest API, Phpoffice.
CDN: Cloudflare CDN, JSDelivr, CDNJS.(nanti Next klo di deploy)
Web Security: Cloudflare WAF, Google reCAPTCHA V2, HSTS, SSL/TLS, Scanning, Encrypted Password, Rate Limiting, XSS Protection, CSRF Protection, dll.
other: Linux, Ngix, Git, Github, SSH, Postman, dll.
Tools: XAMPP, Antigravity, Cursor, Bash, dll.
etc.

        10. Database Notifications: Menyimpan riwayat notifikasi yang bisa dikelola (ditandai baca atau dihapus) melalui dashboard user maupun admin.
